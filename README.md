# Doraemon Todo List

A fun and interactive **Doraemon-themed Todo Application** built using **HTML, CSS, and JavaScript** to help users organize their daily tasks efficiently.

## Features

* Add new tasks
* Mark tasks as completed
* Edit existing tasks
* Delete tasks
* Responsive and clean user interface
* Doraemon-inspired theme and design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Project Structure

```
📁 Todo-App
│── index.html
│── index.css
│── app.js
│── assets/
│   └── background image
```

## How to Run the Project

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Open `index.html` in your browser.
